GitHub 图床的设置

  1. 创建Github账号 和 用来存放图片的仓库 （略）

  2. GitHub申请用来 操作的图片仓库token

    • 进入Github Token 创建页面
    • 勾选 repo 访问权限。然后滚动页面到底部，点击Generate token按钮来生成 Token
    • 复制Token 注意：此 Token 只会显示一次！请务必保存好，否则之后丢失了，就得重新创建了

  3. 配置uPic Github图床

    若存放图片的仓库 是私有的 CDN加速访问 不要勾选
    若存放图片的仓库 是公有的 CDN加速访问 可以勾选

  4. 愉快的使用起来吧
